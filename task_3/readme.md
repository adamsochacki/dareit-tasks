## Task 3

The goal of this task is to run Cloud Function.

Steps:

1. Go to Cloud Function in the Navigation Menu. Choose CREATE FUNCTION.
2. Choose the HTTP as the trigger, leave deafult 256 MB and hit SAVE.
3. In the editor chose Python3.7 and put your code into the main.py file.
4. Copy the url from the Trigger tab and paste it into your browser. It should work.
5. You can modify the code of main.py with python functions that will be triggered when user visit the trigger url.
6. Paste the trigger url to your function in task_3_cloud_function file in your repository and save.
7. Commit and push changes to your remote repository.
